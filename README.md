# 硬件信息
* 仪器型号：Agilent Acqiris TC890
* 系统支持：Windows x86, x64

# Features
* 可视化配置，实时数据处理与显示
* 可同时显示单道计数、自定义符合计数，以及信号到达时间的统计直方图
* 参数均可调，且配置自动保存
* 单道计数与符合计数的数据可保存
